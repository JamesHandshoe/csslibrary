CSS Notes

Purpose of CSS

  • To give the outside of the site a unique look and feel.
  • To alter colors, fonts, backgrounds, etc.
  • To implement some effects and animations, although it's not as powerful as jQuery.

CSS Statements

Four main components include the following:

  1. Declaration: the statement as a whole with curly braces.
  2. Selector: the name of the part you want to style.
  3. Property: properties to edit(e.g., width, height, background, etc.)
  4. Value: numbers, colors, urls - values that we assign.

CSS Setup

You must link the CSS and HTML files so they can 'talk' to each other.

The way to do this is to put a link to the CSS file in the html. A couple things to remember when doing this:

  • Put the link in the head tags of the html file
  • If the CSS file is on the same file level as the HTML file, put the file name in quotes after an href tag.
  • If not, you need to make sure you put the correct file path.
  • After you've linked the files, do a simple test to make sure the link is working.
  • This could be as simple as changing the background color of the body.
